# Notes

## Development Status

`xndbc` is still in development. Release notes will be added as the package
matures.

## Contributing

Open an issue or pull request on the project repository. For local development,
create the development environment and run the test suite:

```bash
conda env create -f docs/environment.yml
conda activate xndbc-dev
pytest
```

## Code of Conduct

Please keep project discussions respectful, constructive, and focused on
improving `xndbc`.
